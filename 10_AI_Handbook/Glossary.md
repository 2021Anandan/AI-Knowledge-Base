\# Glossary



| Term | Definition |

|------|------------|

| Artificial Intelligence (AI) | The simulation of human intelligence by machines to perform tasks such as learning, reasoning, and problem-solving. |

| Machine Learning (ML) | A branch of AI where systems learn patterns from data without being explicitly programmed. |

| Deep Learning | A subset of ML that uses artificial neural networks with multiple layers. |

| Generative AI | AI systems capable of creating new content such as text, images, audio, or code. |

| Large Language Model (LLM) | An AI model trained on vast amounts of text to understand and generate natural language. |

| Prompt | The instruction or question given to an AI model. |

| Prompt Engineering | The practice of designing effective prompts to obtain high-quality AI responses. |

| Token | A unit of text processed by an AI model. |

| Embedding | A numerical representation of text that captures its meaning. |

| Context Window | The amount of information an AI model can consider in a single interaction. |

| Transformer | The neural network architecture used by most modern language models. |

| Fine-tuning | Further training of a pre-trained AI model for a specific task. |

| Hallucination | Incorrect or fabricated information generated by an AI model. |

| Git | A distributed version control system. |

| GitHub | A cloud platform for hosting Git repositories and collaboration. |

| Markdown | A lightweight markup language used for formatting text documents. |

| Ollama | A tool for running Large Language Models locally on a computer. |

| NotebookLM | Google's AI-powered research and note-taking assistant. |

| Gemini | Google's family of multimodal AI models. |

| Responsible AI | The ethical and transparent development and use of AI systems. |
